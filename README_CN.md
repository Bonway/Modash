
Modash（Modern + Dash）是一款一致性、模块化、高性能的实用工具库。

## 特性
Modash 让 Flutter 开发更加简单，降低了操作数组、数字、对象、字符串等数据类型的难度。

## 快速开始
在你的 Flutter 项目的 pubspec.yaml 文件中添加依赖：
```yaml
dependencies:
  modash: latest_version
```
然后运行：
```sh
flutter pub get
```
